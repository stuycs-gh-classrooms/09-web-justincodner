Koch Curve

Basic Koch Curve. koch_curve(leonardo, 4, 20)

This is a Koch Curve, but an extra recursive step is added to make it a Koch Square. Width and Size also decrease as depth increase to ensure that it stays on the screen. koch_curve_modified(leonardo, 2, 20, 10)


    def koch_curve_modified(t, depth, size, width):    
    if (depth == 1):
        t.width(width)
        t.fd(size)
    else:
        koch_curve_modified(t, depth-1, size*.8, width*.8)
        t.lt(90)
        koch_curve_modified(t, depth-1, size*.8, width*.8)
        t.rt(90)
        koch_curve_modified(t, depth-1, size*.8, width*.8)
        t.rt(90)
        koch_curve_modified(t, depth-1, size*.8, width*.8)
        t.lt(90)
        koch_curve_modified(t, depth-1, size*.8, width*.8)
